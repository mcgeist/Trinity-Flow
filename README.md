# Trinity-Flow
Displays historical river flow data in the Trinity river over a three-week timeframe

## Usage:
python main.py YYYY-MM-DD


## Dependencies:
This script requires `hydrofunctions` API for fetching river flow data, `matplotlib` for plotting, `humanize` for data display, and `numpy`/`statistics` for data processing.
```bash
pip install hydrofunctions humanize matplotlib datetime numpy statistics
```
